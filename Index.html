b) Liste des attributs de la balise form

accept La liste des types acceptés par le serveur. Généralement, il s'agit de types de fichier.
accept-charset La liste des jeux de caractères pris en charge.
action L'URI d'un programme qui traite les informations envoyées par le formulaire.
autocomplete Cet attribut indique que ces contrôles ou que ce formulaire peuvent être remplis automatiquement par le navigateur.
enctype Cet attribut définit le type de contenu des données de formulaire envoyées lorsque la méthode utilisée est POST.
method Cet attribut définit la méthode HTTP à utiliser pour l'envoi des données du formulaire (GET par défaut ou POST)
name Le nom de l'élément qui peut être utilisé par le serveur pour identifier le champ d'un formulaire.
novalidate Cet attribut indique que les données du formulaire ne doivent pas être validées lors de l'envoi.
target L'attribut cible spécifie un nom ou un mot-clé indiquant où afficher la réponse reçue après l'envoi du formulaire.
